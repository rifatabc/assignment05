Question-1: What are the different ways to select an element in the DOM?

Anser:There are several ways to select elements from the DOM in JavaScript. Elements are selected based on a specific ID.Multiple elements are selected based on a specific class name.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Anser:You can use the innerHTML property to get or set the inner HTML content of an element. innerText returns only the visually displayed text, which is visible to the user. It excludes parts hidden by HTML tags or styling elements. The textContent property returns only the text and does not affect HTML tags or styles.

Question-3: What is event delegation in the DOM?

Anser:Event delegation is a JavaScript technique where an event listener is attached to a parent and then handles events on its sub-elements through that parent.

Question-4: What is event bubbling in the DOM?

Anser:Event bubbling is a DOM event process where an event starts at a sub-element and gradually reaches its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Anser:The document.createElement() method is used to create a new element. The appendChild() or insertBefore() methods are used to add the newly created element to a parent.
